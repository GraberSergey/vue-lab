# Vue comments lab

Учебный проект из курса по Vue: на странице отображаются комментарии с вложенностью, а также форма для добавления новых сообщений через публичное API.

## Scripts

```sh
# установка зависимостей
npm install

# запуск dev-сервера
npm run dev

# сборка production-версии
npm run build
```

API: `http://95.163.242.125/api`.
